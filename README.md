# 🧮 Algorithm Visualizer

An interactive web-based platform for visualizing algorithms and data structures. Perfect for students, educators, and anyone interested in understanding how algorithms work through dynamic visual representations.

![Algorithm Visualizer Demo](frontPage/image1(2).png)

## 🌟 Features

### 🔄 Sorting Algorithms
- **Selection Sort**: Step-by-step visualization with interactive controls
- **Merge Sort**: Recursive divide-and-conquer visualization
- Real-time array manipulation with animated bars
- Start/Stop/Speed controls for better learning

### 🌳 Data Structures  
- **Binary Search Tree**: Insert, search, and traverse operations
- **Tree Traversal**: Depth-First Search (DFS) with dynamic tree generation
- **Linked List**: Add items and search functionality with visual feedback

### 🎯 Interactive Features
- **Live Preview**: Hover over algorithms on the landing page for instant demos
- **Real-time Animation**: Watch algorithms execute step by step
- **User Input**: Add custom values to data structures
- **Visual Feedback**: Color-coded elements showing algorithm progress

## 🛠️ Tech Stack

- **Frontend**: Vanilla JavaScript, HTML5, CSS3
- **Graphics**: SVG for scalable visualizations
- **Animations**: CSS transitions and JavaScript intervals
- **Architecture**: Modular component-based structure
